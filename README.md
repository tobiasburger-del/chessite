# Chess Companion

A chessgame catalog/analysing website built incrementally as part of a web development course. The project grows week by week — starting from static HTML/CSS and evolving into a full-stack, database-driven, deployed application.

---
 
##  About

This project serves as both a **learning vehicle** and a **real-world chess companion**. Each phase of development aligns with a weekly course topic, allowing hands-on practice with each new concept.

In the end:
Users should be able to upload their games, followed by the ability to analyse them and get a better understanding of their mistakes, and patterns that are most worth practicing.
With the goal of improving their visualised learning curve the most efficient.


##  How to run

npm install,
node server.js,
open http://localhost:3000
